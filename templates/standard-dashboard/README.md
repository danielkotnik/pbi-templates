# Standard Dashboard Template

Place your `.pbit` (Power BI Template) file here.

## How to create a .pbit file

1. Open your base dashboard in Power BI Desktop
2. Go to `File > Export > Power BI Template`
3. Add a description and save as `TEMPLATE_Standard-Dashboard.pbit`

## Template should include

- Scripps lighthouse logo placeholder (top left)
- Standard header layout (dashboard name, tab name, dates, support link)
- Reset Filters button with placeholder bookmark
- User Reference tab (pre-formatted)
- Version Notes tab (pre-formatted)
- Scripps blue color theme applied
