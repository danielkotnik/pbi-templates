# Tab Layout Reference

## Standard Data Tab Layout

```
Row 1: [ Logo ] [ Dashboard Name ]              [ Data Through: ___ ] [ Refreshed: ___ ]
                [ Tab Name       ]              [ Contact: email     ]
Row 2: [ Slicer/Filter 1 ] [ Slicer 2 ] [ Slicer 3 ]        [ RESET FILTERS ]
Row 3: [ Visual A        ] [ Visual B                ]
Row 4: [ Visual C        ] [ Visual D ] [ Visual E   ]
```

## User Reference Tab Layout

```
Row 1: [ Logo ] [ Dashboard Name ]
Row 2: [ Dashboard Overview section ]
Row 3: [ Data Asset ID ] [ Business Owner ] [ Data Sources ]
Row 4: [ Refresh Schedule ] [ Data Lineage ]
Row 5: [ FAQs ]
```

## Version Notes Tab Layout

```
Row 1: [ Logo ] [ Dashboard Name ]
Row 2: [ Version table: Version | Date | Author | Changes ]
```
